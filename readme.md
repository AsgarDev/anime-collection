
# Anime Collection
A SPA about Anime world, built with Symfony and React.

## Run Locally

Clone the project

```bash
  git@github.com:AsgarDev/anime-collection.git
```

Run the docker-compose

```bash
  make first-install
```

Launch the internal server

```bash
  make bash
  symfony serve -d
```
